# RunCat Clone

Another [cute running cat animation on your Windows taskbar](https://github.com/Kyome22/RunCat_for_windows).

### [Download](https://github.com/yehwankim23/runcat-clone/releases/download/v1.0.0/RunCat_Clone_1.0.0.zip)

- Download, extract, and run `RunCatClone.exe`. (Ignore the "_can't be downloaded securely_" warning)
- Right-click `Install.ps1` and **Run with PowerShell** to add RunCat Clone to **computer startup**.
- Right-click `Uninstall.ps1` and **Run with PowerShell** to remove RunCat Clone from **computer startup**.
